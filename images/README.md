# Implementation Images

This directory contains screenshots captured during different stages of the OpenLane RTL-to-GDS implementation flow.

## Images

- **floorplan.png** – Floorplanning stage
- **placement.png** – Standard cell placement
- **cts.png** – Clock Tree Synthesis (CTS)
- **routing.png** – Final routed layout
- **gdsII.png** – Final GDSII layout

These images illustrate the physical evolution of the design throughout the ASIC implementation process.
